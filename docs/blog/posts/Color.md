# 颜色色域全解析：从 SDR 到 HDR 的色彩边界

## 一、什么是色域？

色域（Color Gamut）是指显示设备、图像采集设备或色彩标准所能呈现的颜色范围集合，通常以 **CIE 1931 色彩空间**（国际照明委员会制定的二维色彩坐标系统）为参照基准，用闭合区域直观展示颜色覆盖范围。

简单来说，色域就像画家的调色盘：



* 色域越广，“调色盘” 上的颜色种类越多，能呈现的色彩越丰富、越接近真实世界；

* 色域越窄，可呈现的颜色越有限，可能导致画面色彩单调、失真。

色域的核心价值在于统一色彩呈现标准，确保不同设备（如显示器、电视、打印机）之间的色彩一致性。

## 二、SDR 色域：标准动态范围的色彩基石

### 2.1 定义与核心特性

SDR（Standard Dynamic Range，标准动态范围）是传统显示技术的色彩标准，动态范围（亮度范围）和色域范围相对有限，核心特点是：



* 亮度范围：通常为 0.1\~100 nits（尼特，亮度单位）；

* 色彩表现：仅覆盖人眼可识别色彩的一部分，侧重 “标准化” 而非 “极致还原”；

* 适用场景：传统电视、电脑显示器、网页设计、印刷行业等成熟领域。

### 2.2 主流 SDR 色域标准

#### （1）sRGB



* 发布背景：1996 年由微软和惠普联合制定，是目前最普及的 SDR 色域标准；

* 色域覆盖：在 CIE 1931 色彩空间中覆盖约 35.9% 的可见光谱；

* 应用场景：电脑显示器、笔记本屏幕、手机屏幕（中低端）、网页图片、办公软件（如 Word、Excel）、普通数码照片。

* 优势：兼容性极强，几乎所有电子设备都默认支持，色彩一致性好。

#### （2）Rec.709（ITU-R BT.709）



* 发布背景：1990 年由国际电信联盟（ITU）制定，专为高清电视（HDTV）设计；

* 色域覆盖：与 sRGB 色域范围高度重合（仅在蓝绿通道略有差异），覆盖约 35.9% 可见光谱；

* 应用场景：高清电视节目（720P/1080P）、蓝光光盘、广播电视信号传输。

* 特点：是高清视频领域的 “通用标准”，确保不同电视台、播放设备的色彩统一。

#### （3）Adobe RGB



* 发布背景：1998 年由 Adobe 公司制定，面向专业图像设计；

* 色域覆盖：覆盖约 50.6% 可见光谱，重点扩展了青绿色系（适合自然风景、植物摄影）；

* 应用场景：专业摄影后期、印刷设计、广告制作。

* 优势：比 sRGB 更接近印刷色域，避免设计稿在印刷后出现色彩偏差。

## 三、HDR 色域：高动态范围的色彩突破

### 3.1 定义与技术优势

HDR（High Dynamic Range，高动态范围）是新一代显示技术的核心标准，通过扩展 **亮度范围** 和 **色域范围**，实现更接近人眼视觉体验的色彩呈现，核心优势：



* 亮度范围：峰值亮度可达 400\~10000 nits，支持更暗的暗部（0.0001 nits 以下）和更亮的高光；

* 色彩表现：覆盖更广泛的可见光谱，能呈现 SDR 无法显示的 “极致色彩”（如深海蓝、日落橙）；

* 核心逻辑：还原真实世界的 “明暗对比” 和 “色彩层次”，让画面更具沉浸感。

### 3.2 主流 HDR 色域标准

#### （1）Rec.2020（ITU-R BT.2020）



* 发布背景：2012 年由 ITU 制定，专为超高清电视（UHD/4K/8K）设计；

* 色域覆盖：覆盖约 75.8% 可见光谱，是目前色域最广的标准之一，包含红、绿、蓝三原色的极致延伸；

* 应用场景：8K 电视、专业超高清视频制作、未来显示技术（如 Micro LED）；

* 现状：目前多数消费级设备无法完全覆盖 Rec.2020（仅高端电视能覆盖 80% 以上），主要作为 “未来标准” 预留。

#### （2）DCI-P3



* 发布背景：由数字电影倡导联盟（DCI）制定，最初用于数字影院；

* 色域覆盖：覆盖约 54.3% 可见光谱，重点优化了红色和绿色系（适合电影色彩还原）；

* 应用场景：


  * 影院版：数字电影放映机（100% 覆盖）；

  * 消费级：高端电视（如 OLED）、手机（如 iPhone、旗舰安卓机）、专业显示器、4K 蓝光光盘；

* 优势：兼顾 “影院级色彩” 和 “消费级设备适配”，是目前 HDR 普及度最高的标准。

#### （3）HDR10/HDR10+ 关联色域



* HDR10：是 HDR 的基础标准（免费开源），默认采用 DCI-P3 或 Rec.2020 色域，要求峰值亮度≥1000 nits；

* HDR10+：HDR10 的升级版本，支持动态元数据（逐帧调整亮度和色域），进一步优化色彩还原；

* 应用：主流中高端电视、流媒体平台（如亚马逊 Prime Video）。

#### （4）Dolby Vision



* 发布背景：由杜比实验室制定，闭源收费标准；

* 色域特性：基于 Rec.2020 色域，支持更高的峰值亮度（最高 10000 nits）和 12 位色深（4096 级色彩梯度）；

* 应用场景：高端电视、电影制作、流媒体（如 Netflix、Disney+）；

* 优势：色彩还原精度最高，是专业级 HDR 的代表。

## 四、SDR 与 HDR 色域核心对比



| 对比维度      | SDR 色域                       | HDR 色域                        |
| --------- | ---------------------------- | ----------------------------- |
| 亮度范围      | 0.1\~100 nits                | 0.0001\~10000 nits（峰值）        |
| 色域覆盖（CIE） | 35.9%\~50.6%（sRGB/Adobe RGB） | 54.3%\~75.8%（DCI-P3/Rec.2020） |
| 色深支持      | 8 位（256 级色彩梯度）               | 10\~12 位（1024\~4096 级色彩梯度）    |
| 色彩表现      | 满足基础色彩需求，部分深色 / 亮色丢失         | 还原真实色彩，明暗细节丰富，色彩过渡自然          |
| 设备要求      | 普通显示面板，成本低                   | 高动态范围面板（如 OLED、Mini LED），成本高  |
| 内容支持      | 所有传统视频、图片、文档                 | 需 HDR 专用内容（如 HDR 电影、HDR 游戏）   |

## 五、总结

色域是显示技术的核心指标之一，SDR 色域奠定了传统显示的 “标准化基础”，适合日常办公、普通娱乐等场景；HDR 色域则通过 “宽色域 + 高动态范围” 的双重突破，实现了从 “能看到色彩” 到 “还原真实色彩” 的跨越，成为高端显示设备（如 4K 电视、专业显示器）的核心竞争力。

随着 8K 技术、Micro LED 等显示技术的发展，色域将进一步向 “100% 可见光谱” 逼近，未来我们将能在屏幕上看到更接近真实世界的色彩体验。
